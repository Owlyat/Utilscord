# Utilscord
 
## A TUI based program for live show in pure rust

### Features
Tabs|Shortcuts
--- | ---
Change tab | Shift + Arrow left/Right
Navigate | Shift + Arrow Up/Down
Quit | Ctrl + C / Ctrl + Q

| SoundPlayer | Shortcuts |
|:-----------:|:---------:|
|play| Enter |
|pause| Space |
|stop| Backspace |
|volume up| Shift + Up Arrow |
|volume down| Shift + Down Arrow |
| Exit | Esc |
